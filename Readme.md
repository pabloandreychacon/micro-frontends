# How to Build Micro Frontends in React with Vite and Module Federation
This tutorial will guide you through the process of building micro frontends in React using Vite and Module Federation. Micro frontends allow you to break down a large frontend application into smaller, manageable pieces that can be developed, deployed, and maintained independently.
